# 🌟 YouthInAction

📌 Overview

# YouthInAction is a full-stack web application built to help communities (especially barangays) manage volunteer programs efficiently. It allows organizers to create events and enables users to participate, track, and engage in meaningful activities.

# 🚀 Features
- 👤 User Authentication (Login & Registration)
- 📅 Event Creation and Management
- 📝 Volunteer Registration for Events
- 📊 Event Tracking and Participation Monitoring
- 🔎 Public Event Viewing (even without login)

# 🛠️ Tech Stack
Frontend:
- HTML/CSS/JS
- React + Vite

Backend:
- Node.js

# Database & Services

- Firebase Authentication – for user accounts (login, register, logout)
- Supabase – for image storage (event images and uploads)

##🧠 System Overview
# YouthInAction uses a separated service architecture:

- Firebase handles secure user authentication
- Supabase handles image storage for event-related media
- Node.js + Express manages server-side logic and API requests
  
## ⚙️ Installation\
# Clone repository
git clone https://github.com/your-username/youthinaction.git

# Go into project folder
cd youthinaction

# Install dependencies
npm install

# Start development server
npm run dev
